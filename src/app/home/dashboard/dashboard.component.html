Hola, bienvenida/o a Attia App:

Introducción

Attia App es una aplicación diseñada para gestionar algunos campos de una empresa (empleados, gatos, ganancias...), de la manera más optima. Para ello hemos implementado interfaces agradables para el usuario, de forma que pueda ver la información de la forma más clara y con la mayor rapidez posible.

Objetivos

El objetivo de esta aplicación es que el usuario sea capaz de adaptarse rapidamente a la interfaz que presenta y que su uso sea lo mas fluido posible. Para ello hemos intentado crear un interfaz lo más intuitivo posible, también queríamos algo simple y que no sea muy recargado para que su uso sea fácil.

Tecnologías

Hemos usado las que creemos que son las herramientas mñas utiles para lo que buscabamos con esta aplicación. Utilizamos Angular como framework, bootstrap junto con css para los estilos y sweetalert2 para las notificaciones.
